    ClampToNegative(Double)

    If the value is non-positive it is returned as is, otherwise a zero is returned. A more apt signature would be ClampToNonPositive, but this is a bit too verbose and potentially confusing for some. WARNING: Never use this extension like this: 1.0.ClampToNegative(), since if the value is negative the sign (if there is one) will not be passed to the extension method leading to unexpected results. For example this, -1.0.ClampToNegative() is interpreted by the extension method like this 1.0.ClampToNegative()! You could use parentheses but to be on the safe side avoid this usage.
